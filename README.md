# <a href="#"><img height="32" alt="icon" src="WU Killer.png" /></a> WU Killer

Windows Update のコンポーネントを強制的に無効化します。

Force disable Windows Update components.

## ダウンロード / Download

[**ここから / Here**](https://github.com/s1204IT/WU-Killer/releases/latest/download/WU_Killer.exe)

## 使い方 / How to use

<a href="#"><img width="752" height="490" alt="image" src="screenshot.png" /></a>

- 【還】：再有効化 (Re-enable)
- 【殺】：無効化 (Disable)
